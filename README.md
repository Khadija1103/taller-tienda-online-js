# Taller Tienda Online JS

## Descripción

Proyecto desarrollado en JavaScript y Node.js que implementa conceptos de Programación Orientada a Objetos (POO), herencia y modularización mediante la creación de un sistema básico de tienda online.

## Estructura del Proyecto

```text
tienda/│
usuario.js
producto.js
index.js
README.md
```

## Funcionalidades

### Gestión de Usuarios

* Clase `Usuario`
* Clase `UsuarioVIP` que hereda de `Usuario`
* Visualización de información del usuario
* Gestión de beneficios para usuarios VIP

### Gestión de Productos

* Clase `Producto`
* Clase `ProductoDigital` que hereda de `Producto`
* Visualización de información de productos
* Gestión de características adicionales para productos digitales

## Tecnologías Utilizadas

* JavaScript
* Node.js
* Visual Studio Code
* Git y GitHub

## Conceptos Aplicados

* Clases y objetos
* Constructores
* Uso de `this`
* Herencia con `extends`
* Uso de `super()`
* Modularización
* `module.exports`
* `require()`

## Ejecución del Proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/Khadija1103/taller-tienda-online-js.git
```

2. Ingresar a la carpeta del proyecto:

```bash
cd taller-tienda-online-js
```

3. Ejecutar el programa:

```bash
node index.js
```

## Resultado Esperado

El sistema mostrará en consola la información de usuarios y productos, incluyendo las características especiales de usuarios VIP y productos digitales.
