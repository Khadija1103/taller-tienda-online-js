# Taller Tienda Online JS

## Descripción

Proyecto desarrollado en JavaScript y Node.js aplicando Programación Orientada a Objetos (POO), herencia y módulos.

## Archivos del proyecto

* `usuario.js`: Clases `Usuario` y `UsuarioVIP`.
* `producto.js`: Clases `Producto` y `ProductoDigital`.
* `index.js`: Importa y ejecuta el sistema.

## Ejecución

```bash
node index.js
```

## Conceptos aplicados

* Clases y objetos
* Constructores
* Herencia (`extends`)
* `super()`
* `module.exports`
* `require()`
